L'association URSOL propose à un public adulte une écoute, un accompagnement dans les démarches administratives, des petits-déjeuners, des repas, l’entretien du linge, douches et toilettes. Ayant de gros soucis de praticité lors de l'élaboration des démarches admninistratives destinées aux personnes dans le besoin, ils ont décidés de nous contacter (un camarade de classe et moi) afin de résoudre ce problème. Durant une période de 2 stages de 6 semaines, nous avons élaboré ce logiciel qui permet de creer un dossier pour chaque personne qui va contenir toutes les informations nécessaires pour que les bénévoles puissent en amont ou plus tard faire la démarche à leurs places (nom, date de naissance,déclaration d'imposition, numéro de sécurité sociale..). Rémunération : 400€.

    Compétences:
  • Utilisation de la méthode Merise
  • Créer une base de données convenant aux attentes de l'organisation.
  • Installer une base de données MySQL sur un serveur local.
  • Manipuler des données récupérées d'une base de donnée MySQL en C#.
  • Insérez des données récupérées d'une application dans une base de données MYSQL en C#.
  • installation d’un environnement FTP sur un serveur NAS.
  • Utilisation du protocole FTP pour transmettre et récupérer des fichiers à un serveur.
  • Travailler en mode projet (Git, Trello).
