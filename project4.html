Au cours de ma formation, j'ai travaillé sur un projet d'API REST qui comprenait deux parties. La première partie était une API REST développée en utilisant NodeJS et une base de données MongoDB. La seconde partie était une API REST développée en utilisant Symfony et une base de données MySQL. Les deux API ont été conçues pour être sécurisées grâce à l'utilisation de tokens JWT pour l'authentification. Les deux APIs avaient également pour objectif de fournir une expérience utilisateur fluide et une réponse rapide aux requêtes grâce à une bonne gestion des erreurs.

    Compétences:
  • Développement d'API REST en utilisant NodeJS et Express
  • Utilisation de Mongoose pour interagir avec une base de données MongoDB
  • Développement d'API REST en utilisant Symfony
  • Utilisation de MySQL pour la gestion de base de données relationnelles
  • Utilisation de JWT pour l'authentification des utilisateurs
